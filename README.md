# Artillery

## Load testing simple Go and Node CRUD apis
Both were hosted on a nanode on linode. 
Both had a local postgresql client to connect to.


![Linode details](images/nanode-details.png)

Framework: fiber
ORM: Gorm

### Node api
Framework: Express
ORM: Drizzle
